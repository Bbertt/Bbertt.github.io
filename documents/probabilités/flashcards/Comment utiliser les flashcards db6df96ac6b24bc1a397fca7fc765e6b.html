Les flashcards favorisent l’apprentissage actif. Elles sont également un moyen ludique de réviser son cours. Cependant, réviser les flashcards ne suffit pas à préparer le contrôle continu ou le contrôle terminal. Par ailleurs, lorsque les réponses des cartes ne sont pas comprises, il est essentiel de retourner consulter son cours.

Une fois les cartes découpées, l’idée générale est de réviser plus souvent les cartes les plus difficiles, et de temps en temps les cartes faciles. Vous pouvez distinguer plusieurs piles de cartes : les cartes à consulter tous les jours, les cartes à consulter tous les deux jours, les trois jours etc. C’est la méthode proposée dans la vidéo accessible par ce lien.

En utilisant le logiciel Anki

Anki est un logiciel libre permettant d'apprendre et de réviser des cartes-mémoires grâce à la répétition espacée. source : Wikipedia

L’avantage d’Anki c’est qu’il adapte les cartes à faire réviser en fonction des difficultés : on peut rapidement enterrer les cartes les plus faciles, et faire ressortir plus souvent les cartes difficiles. L’application sur ordinateur est gratuite. Il existe des clients mobiles, pour faire tourner l’application sur votre téléphone, mais ils ne sont malheureusement pas tous gratuits. Sur mobile, je vous conseille donc plutôt la version web.

Dans tous les cas vous avez besoin d’un compte sur le site Ankiweb : https://ankiweb.net/about . Une adresse email et un mot de passe suffisent. Vous aurez besoin de charger les paquets de carte sur ordinateur la première fois, mais vous pouvez vous passer d'ordinateur ensuite si vous avez un smartphone ou une tablette.

Utiliser le logiciel sur ordinateur pour importer les cartes

Allez sur le site https://apps.ankiweb.net/ pour télécharger et installer le logiciel sur ordinateur. Téléchargez le fichier qui contient toutes les cartes à mémoriser : en cliquant sur ce lien. Après installation, lancez le logiciel, connectez-vous et choisissez “ajouter” dans la fenêtre principale. Parcourez les dossier de votre ordinateur pour trouver le fichier .apkg que vous venez de télécharger. A la fin de la procédure, vous devez voir apparaître les cartes “Probabilités 1”. Synchronisez le tout afin de pouvoir visualiser les paquets depuis https://ankiweb.net/decks .

Accéder aux cartes depuis un téléphone ou une tablette

Vous aurez besoin d’accéder au moins une fois à l’application sur ordinateur afin d’importer le paquet de cartes (voir les instructions dans la section précédente). Ensuite, vous pouvez vous passer d’ordinateur si vous le souhaitez.

Sous Android

Il existe une application gratuite sous Android, ou bien vous pouvez utiliser la version Web depuis votre navigateur internet.

Sur iPhone/IPad

Malheureusement, l’application sur l’Applestore est payante et à mon sens, beaucoup trop chère. Vous pouvez en revanche utiliser la version web gratuitement lorsque vous êtes connectés à internet. Ouvrez https://ankiweb.net/decks dans Safari, cliquez sur l’icone de partage de lien puis déroulez les options et cliquez sur le ➕ “Sur l’écran d’accueil”. Cela crée sur votre écran d’accueil un raccourci vers la page. Cela ressemblera beaucoup à une application, en revanche vous aurez besoin d’une connexion internet, donc assurez-vous d’avoir une connexion wifi ou un forfait illimité pour internet afin de ne pas avoir de coûts supplémentaires.
